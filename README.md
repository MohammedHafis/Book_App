# 📚 Book Management API

A simple RESTful Book Management API built using Java and Spring Boot. This application allows users to create, retrieve, and delete books through REST endpoints.

---

## 🚀 Features

- ➕ Add a new book
- 📖 Get all books
- 🔍 Get a book by ID
- ❌ Delete a book by ID
- RESTful API architecture
- Spring Data JPA integration

---

## 🛠️ Technologies Used

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- H2 Database / MySQL

---

## 📌 API Endpoints

| Method | Endpoint | Description |
|----------|----------|-------------|
| POST | `/api/books` | Create a new book |
| GET | `/api/books` | Get all books |
| GET | `/api/books/{id}` | Get book by ID |
| DELETE | `/api/books/{id}` | Delete book by ID |
